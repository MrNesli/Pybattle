# Pybattle
A python ascii text art pokemon style game in the terminal using ANSI escape codes.

Example of ascii text art:
```
╭─ BEDROOM ─┬──────────────────╮
│   ╰───────╯       ||||       │
│                   ||||       │
│                     ─┬─┬─┬─┬─┤
│                              │
│                              │
│╭│╮   ╶─╮                     │
││││    ░│                     │
│╰│╯   ╶─╯           ╭─────┬─╮ │
│                    │░░░░░│▓│ │
│                    ╰─────┴─╯ │
╰──────────────────────────────╯
```

# Installation:
**Linux:**

```$ pip install --upgrade setuptools ```

```$ python3 -m venv env```

```$ source env/bin/activate```

```$ pip install . ```

```$ python main.py ```
